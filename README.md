This repository contains the interactive portfolio website of Satwik Shreshth, an MCA student at Sikkim University, featuring a responsive and animated web interface. The website showcases academic background, technical skills, experiences, and contact information.
Features
Visually engaging animated background and loading screen
Responsive design for desktops and mobile devices
Quick navigation header with smooth scrolling
Highlighted sections: About, Skills, Experience, Contact
Dynamic grids for skills and experience powered by JavaScript
Social links and quick action buttons (Email, WhatsApp, Resume, etc.)
Special effects including intersection observer animations and parallax effects
Easter eggs (Konami code surprise)
Modern glassmorphism card UI
Light/dark gradients and custom fonts for clean look
Technologies Used
HTML5: Semantic structure and modern tags

CSS3: Animations, gradients, grid/flex layouts, glassmorphism effects, custom properties

JavaScript (Vanilla): DOM manipulation, Intersection Observer, canvas-based particle system, dynamic rendering of skills/experience/social links

Google Fonts: Inter, JetBrains Mono, Playfair Display
